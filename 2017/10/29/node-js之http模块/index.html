http.ClientRequest 是http.request或者http.get返回产生的对象，表示一个已经产生而且正在进行的http请求，提供一个response事件，也就是我们使用http.get和http.request方法中的回调函数所绑定的对象，我们可以显示的绑定这个事件的监听函数。
