Node.js 获取req 参数的三种方式

express获取参数有三种方法：官网介绍如下

  • Checks route params (req,params), ex: /user/:id
  • Checks query string params (req.query), ex: ?id=12
  • Checks urlencoded body params (req,body), ex: id=

第一种情况：req.params

浏览器url：http://127.0.0.1:9002/server/info/deleteServiceById/6b2f864f-05dd-4b2b-8743-ebed1537eb2f，服务器端代码

第二种情况：req.query

浏览器url：http://127.0.0.1:9002/server/info/updateParamById?id=55f8127a-6dcb-4e91-9507-17b2284a0c71&paramName=EditgoodsCount&paramType=int&paramExplain=(修改)返回商品价格

第三种情况：req.body
