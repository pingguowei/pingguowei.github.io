什么是SQL语句

sql语言：结构化的查询语言。（Structured Query Language），是关系数据库管理系统的标准语言。

它是一种解释语言：写一句执行一句，不需要整体编译执行。
语法特点：

  1. 没有“ ”，字符串使用‘ ’包含
  2. 没有逻辑相等，赋值和逻辑相等都是=
  3. 类型不再是最严格的。任何数据都可以包含在‘ ’以内
  4. 没有bool值的概念，但是在视图中可以输入true/false
  5. 它也有关系运算符：> < >= <= = <> != ,它返回一个bool值
  6. 它也有逻辑运算符： ！(not) &&(and) ||(or)
  7. 它不区别大小写
